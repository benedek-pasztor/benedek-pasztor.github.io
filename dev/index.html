What is this about?

Here you will be able to interactively create deep insights of COVID19 data, e.g. confirmed cases or mortality rate. You will be able to compare the spread of the virus in different countries, or see how the virus is evolving within a country. See some example charts by clicking on the pictures below!

How to use?

Step 1: Select the figure and adjust filters on the top of the dashboard

On the top row will always be able to choose the figure you would like to see. You can check its absolute value, or how much it has changed since yesterday or since last week. You can also adjust any filter to see only one continent's data or to compare specific countries. You will be able to compare countries, and obtain interesting charts interactively.



Step 2: Select one of the following views

Top countries

Top countries of the figure and the filters you have selected on the top row.

Map view

Colored map of the figure and the filters you have selected on the top row.

Days since ... comparison

How the selected figure has been evolving since confirmed cases have passed 100

Scatter (XY) plots

Relation of the selected figure and a customizable second figure on the X axis.



Step 3: Adjust further parameters on the bottom

On the bottom there are view-specific filters and parameter settings. For example, you can set the date to see, on the 'Top countries' view you can set the number of top countries you would like to see, on 'Days since ... comparison' view you can change the threshold default value (100) to other, and on the 'Scatter (XY) plots' view you can set what you would like to see on the X axis.



Data sources: John Hopkins University, World Bank, WHO
